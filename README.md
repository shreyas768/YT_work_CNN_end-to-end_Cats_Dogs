# YT_work_CNN_end-to-end_Cats_Dogs

This repository contains a complete end-to-end Convolutional Neural Network (CNN) project for image classification (Cats vs Dogs), built using PyTorch.

The project is part of my YouTube series where I walk through the entire pipeline — from data loading to model training and evaluation.
